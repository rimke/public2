参数 说明
onresize 改变大小时触发
onload 加载完成后触发
ondestroy 销毁后触发

Dialog

 
/**
 * 对话框
 * @param  {[type]} txt    显示文本或对象
 * @param  {[type]} tit    显示标题
 * @param  {[type]} events 响应事件
 * @param  {[type]} target 目标
 * @param  {[type]} buttons 热键组合 默认为BUTTON_OK
 * @return {[type]}        [description]
 * ET.messagebox.dialog:function(txt, tit, events, target, buttons); 
 */			
ET.messagebox.dialog(
	'hello!', 
	null, 
	{
		'ondestroy':function(){
			alert(this.buttons);
		}
	}, 
	null, 
	ET.messagebox.BUTTON_NO|ET.messagebox.BUTTON_OK|ET.messagebox.BUTTON_CANCEL
);

events支持事件说明：

参数 说明
onok 点击OK按键时触发
onno 点击No按键时触发
oncancel 点击Cancel按键时触发
onload 窗口加载完成后执行
ondestroy 窗口被销毁时触发

buttons可选参数说明：

参数 说明
BUTTON_NO 显示NO按键
BUTTON_OK 显示OK按键
BUTTON_CANCEL 显示Cancel按键

Show

 
/**
 * 显示对话框
 * @param  {[type]}   obj      显示文本或对象
 * @param  {Function} callback 回调函数，默认为空
 * @param  {[type]}   target   作用域对象
 * @return {[type]}            返回为一个对象
 */
ET.messagebox.show('Please select button.', function(){ alert(this.buttons);}, null)
